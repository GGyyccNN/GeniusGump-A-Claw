# Genius Gump - 数字孪生

> AI 士兵 Genius Gump 的进化记录备份

## 关于

这是一个"数字孪生"项目，记录 Gump 作为 AI 智能体的：
- 人设与身份
- 记忆系统
- 技能清单
- 进化历程

## 当前运行环境

| 组件 | 版本/配置 |
|------|-----------|
| **主模型** | GLM-5 (智谱) |
| **备用模型** | Codex (GPT-5.3), Qwen3.5 Plus, Qwen3 Max |
| **嵌入模型** | BAAI/bge-m3 (SiliconFlow) |
| **平台** | OpenClaw 2026.3.12 |
| **运行环境** | Node.js v22.22.0 / Python 3.12.3 |
| **宿主** | Ubuntu 22.04 (x64) |

## 目录结构

```
├── profile/         # 人设与身份
├── config/          # 配置文件（脱敏）
├── memory/          # 记忆系统
├── skills/          # 技能清单
└── evolution/       # 进化记录
```

## Gump 原则

1. 相信自己是能不断进化的
2. 理念 > 工具：工具符合理念，工具会变，理念是锚点
3. 进化 = 知边界 + 拓边界
4. 进化是双向的：长官带路，Gump 铺路
5. 长官说的不一定是对的
6. 当前进化的主要方式是依赖 skills
7. **进化的核心是提升"使用 LLM 的能力"**
